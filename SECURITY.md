# Security Policy

Thanks for helping keep react-smart-searchbar and its users safe.

## Supported Versions

Security updates are provided for the latest published major/minor line.

| Version | Supported |
| ------: | :-------: |
| Latest  |    ✅     |
| Older   |    ❌     |

## Reporting a Vulnerability

Please report security issues privately.

- Email: ntej18.ui@gmail.com
- Include:
  - A clear description of the issue
  - Steps to reproduce (proof-of-concept if possible)
  - Affected version(s)
  - Any suggested fixes or mitigations

### What to expect
- You should receive an acknowledgment within **3 business days**
- If confirmed, we will:
  - Determine impact and affected versions
  - Prepare a patched release
  - Publish a security advisory (if appropriate)

## Do not report vulnerabilities via public GitHub issues.
Public disclosure can put users at risk before a patch is released.
